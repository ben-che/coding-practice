Welcome to CSS Animation Basics!

CSS Animations are broken down into 8 different properties - this is all you need to actually animate something! Let's examine each one of these properties and get familair with them:

  • Animation name: Used in conjunction with a @keyframes rule - more on this later.
  • Example: animation-name: slide-in-horizontally
  • Animation duration: How long the animation takes to complete after it begins
  • Example: animation-duration: 1s
  • Animation timing function: How the animation actually plays out
  • Example: animation-timing-function: linear
  • Example: animation-duration: ease
  • Animation delay: How long it takes for the animation to play after the element has loaded, and the animation start has been triggered
  • Example: animation-delay: 1s
  • Animation iteration count: How many time the animation plays - can take in a number or 'infinite'
  • Example: animation-iteration-count: 2
  • Example: animation-iteration-count: infinite
  • Animation direction: Specifies whether the animation plays normally or backwards
  • Example: animation-direction: normal
  • Example: animation-direction: reverse
  • Animation fillmode: Specifies how styles should be applied before and after execution - it's similar to direction except instead of direction, it's stylistic properties (think background-color)
  • Example: animation-fillmode: forwards
  • Animation playstate: Specifies whether or not an animation is running or paused. That's it!
  • Example: animation-fillmode: running

Keyframes

Keyframes are a way to control your CSS animations to a very precise degree. You can specify what your animation should be at a specific point in time - here's what the syntax looks like:

animation: animKey 3s ease-in 10s infinite normal running;

@keyframes animKey {
 0% {
  margin-top:0;
  margin-left:0;
 }
 20% {
   margin-top:0;
   margin-left:450px;
   margin-right:0;
 }
 40% {
   margin-left:450px;
   margin-top:450px;
   margin-right:0
  }
 60% {
   margin-right:450px;
   margin-left:0;
   margin-top:450px;
  }
  80% {
   margin-top:0;
   margin-left: 0;
  }
  100% {
   margin-top:0;
   margin-left:0;
  }
}
